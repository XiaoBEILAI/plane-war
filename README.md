# 飞机大战 (Plane War Game)

一个使用 HTML5 Canvas 和 JavaScript 开发的经典飞机大战游戏。

## 在线游玩

[点击这里](https://你的用户名.github.io/plane-war) 在线游玩（需要先将仓库发布到 GitHub Pages）

## 功能特性

- 🎮 流畅的鼠标控制
- 🔫 自动射击系统
- ⭐ 动态星星背景
- 💥 爆炸特效
- 📈 等级系统（随着分数提升，敌机变得更强大）
- 🏆 本地最高分记录
- ❤️ 生命系统

## 操作说明

- **鼠标移动**：控制飞机移动
- **自动射击**：飞机会自动发射子弹
- **躲避敌机**：避免与敌机碰撞

## 本地运行

1. 克隆仓库或直接下载 `index.html` 文件
2. 用浏览器打开 `index.html`
3. 点击"开始游戏"按钮

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

## 部署到 GitHub Pages

1. 将此仓库推送到你的 GitHub
2. 进入仓库 Settings -> Pages
3. 选择 `main` 分支作为源
4. 访问生成的 URL 即可游玩
